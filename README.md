# TaskMind 🧠

O TaskMind é um projeto simples de gerenciamento de tarefas que eu desenvolvi com o objetivo de praticar lógica de programação e desenvolvimento web. A ideia foi criar algo além de uma lista comum, trazendo algumas funcionalidades que ajudam na organização e produtividade do usuário.

O sistema permite adicionar tarefas com nome, data e nível de prioridade, além de organizar automaticamente essas tarefas de acordo com o prazo. Também inclui sugestões de qual tarefa fazer primeiro e alerta quando existem tarefas atrasadas.

Esse projeto foi feito pensando em treinar conceitos básicos, mas tentando aplicar uma lógica um pouco mais “inteligente”, simulando decisões automáticas dentro do sistema.



## Como funciona

O funcionamento é bem direto:

Você digita o nome da tarefa, escolhe uma data e define a prioridade. Depois de adicionar, a tarefa aparece na lista.

O sistema analisa a data e pode ajustar a prioridade automaticamente dependendo de quanto tempo falta para o prazo.

Também existe um botão que sugere qual tarefa você deveria começar primeiro, baseado na data mais próxima.

Se houver tarefas atrasadas, o sistema mostra um alerta avisando.

As tarefas ficam salvas no navegador, então mesmo se fechar a página elas continuam lá.



## Funcionalidades

* Adicionar tarefas com nome, data e prioridade
* Marcar tarefas como concluídas
* Atualização automática de prioridade
* Alerta de tarefas atrasadas
* Sugestão de próxima tarefa
* Salvamento automático no navegador



## Tecnologias utilizadas

* HTML para a estrutura
* CSS para o visual
* JavaScript para a lógica


## Como executar o projeto

1. Baixe ou clone este repositório
2. Abra a pasta no Visual Studio Code
3. Abra o arquivo index.html no navegador

Se quiser uma experiência melhor, pode usar a extensão Live Server no VS Code.
